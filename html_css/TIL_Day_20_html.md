# html 기본 문법

## 태그
- html에서 태그(tag)는 가장 중요하다.
- 해당 문자열 등의 자료를 태그로 감싸면, 해당 태그의 목적에 맞게 정보가 변형된다.
- 태그는 열리고 닫힌다. <열림 태그>내용<\/닫힘 태그> 닫히는 태그는 앞에 /가 붙는다.
- \<strong>강조\</strong>
- <strong>강조</strong>
- 태그는 중첩해서 사용할 수 있다.
