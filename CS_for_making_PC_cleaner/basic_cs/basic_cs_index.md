1.1 **컴퓨터 시스템의 기본 구조**
   - 1.1.1 하드웨어 기초
     - CPU (중앙 처리 장치)

     - 저장 장치 (HDD, SSD)
     - 입력/출력 장치 (키보드, 마우스, 모니터)
   - 1.1.2 소프트웨어 구성 요소
     - 운영 체제
     - 응용 프로그램
     - 드라이버
   - 1.1.3 컴퓨터 아키텍처
     - 폰 노이만 아키텍처
     - 현대 컴퓨터 아키텍처의 발전

   - 1.2.1 운영 체제의 역할
